# Example Mermaid

You can add diagrams using [mermaid](https://mermaidjs.github.io/#/) e.g.:

````
```mermaid
graph LR
install[Install Theme]
install --> configure[Configure]
configure --> draw[Write Docs With Diagrams]
```
````

gives:

```mermaid
graph LR
install[Install Theme]
install --> configure[Configure]
configure --> draw[Write Docs With Diagrams]
```
